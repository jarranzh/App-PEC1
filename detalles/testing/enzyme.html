ENZYME

¿Qué es Enzyme?

Enzyme es una utilidad para realizar pruebas en JavaScript para React que facilita el testing de componentes React. Se puede utilizar con ejecutores de tests como Mocha/chai, jasmine, jest, should o expect.

Para instalar Enzyme hay que ejecutar el siguiente comando: 


            npm i --save-dev enzyme enzyme-adapter-react-16
