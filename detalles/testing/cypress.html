CYPRESS

¿Qué es Cypress?

Cypress es un ejecutor de tests gratis, de código abierto e instalado localmente. Sirve para configurar y escribir tests mientras desarrollas tu aplicación localmente, siguiendo con la filosofía TDD (Test-Driven Development) Permite realizar tests end-to-end, tests unitarios y tests de integración. Con Cypress puedes hacer tests sobre cualquier cosa que se ejecute en un navegador.

Lo que hace único a Cypress es lo siguiente:

Arquitectura

La mayoría de herramientas de testing operan ejecutando fuera del navegador y ejecutando comandos remotos a través de la red. Cypress se ejecuta en el mismo bucle de ejecución que la aplicación.

Aceso Nativo

Cypress opera dentro de tu aplicación, lo que significa que tiene un acceso nativo a cada objeto.

Nueva clase de testing

Cypress te permite alterar cualquier aspecto de cómo funciona tu aplicación

Atajos

Cypress evita que siempre tengas que actuar como un usuario para generar el estado de una situación concreta. Con Cypress puedes interactuar y controlar tu aplicación programáticamente.

Depuración

Cypress se ha construido por usabilidad. Hay cientos de mensajes de error personalizados describiendo la razón exacta por la que el test ha fallado
