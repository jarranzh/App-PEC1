Rollup

¿Qué es Rollup?

Rollup es un empaquetador de módulos para JavaScript que compila pequeños trozos de código en algo más grande y complejo, como una librería o aplicación. Utiliza el nuevo formato estandarizado para módulos de código incluido en la revisión ES6 de JavaScript.
