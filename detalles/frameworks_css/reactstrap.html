REACTSTRAP

¿Qué es REACTSTRAP?

Componentes de React Bootstrap 4 fáciles de usar.

Reactstrap está disponible como un paquete npm. Para instalarlo y guardarlo en las dependencias de tu package.json, ejecuta: 


            npm install --save reactstrap react react-dom

Luego puedes instalar los componentes que necesites como este:

Ejecutando el siguiente comando:


            import React from 'react';
            import { Button } from 'reactstrap';
            
            export default (props) => {
                return (
                <Button color="danger">Danger!</Button>
                );
            };
